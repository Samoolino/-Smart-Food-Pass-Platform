# VICTUALS Web Marketing

This app is the first public-facing VICTUALS frontend scaffold.

## Purpose

It introduces VICTUALS as a Smart Nutritional Food Plan Pass platform with:

- multi-market merchant selection
- $WO in-app value display
- plan and package explanation
- product and nutritional intelligence overview
- merchant network onboarding
- contract-backed proof cards
- West Africa pilot positioning

## Initial views

- Home
- How It Works
- Market Stores
- Plans and Packages
- Product and Nutritional Intelligence
- Merchant Network
- Contract Overview

## Palette source

Use the palette defined in:

`docs/design/victuals-ui-palette.md`

Core accents:

- forest for trust and primary brand
- leaf for approved nutrition and merchant states
- gold for $WO and package value
- amber for pending or review states
- red for failed or blocked states
- blue for contract proof
- cyan for API and store integration
